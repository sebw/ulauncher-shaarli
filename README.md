# Shaarli Bookmarks Search

Search your bookmarks stored in Shaarli.

## Requirements

Ulauncher: [https://ulauncher.io](https://ulauncher.io)

Shaarli Library: `pip install shaarli-client`

## Limitations

For now, search is only tag based.
